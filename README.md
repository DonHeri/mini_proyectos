# mini_proyectos

Colección de pequeños proyectos diseñados para practicar Python, explorar librerías, reforzar conceptos fundamentales y experimentar con diferentes ideas.

El objetivo es **aprender construyendo**, iterar rápido y dejar registrado lo que voy dominando en cada etapa.

## Objetivo

* Practicar Python desde cero hasta niveles más avanzados.
* Explorar conceptos útiles para desarrollo.
* Desarrollar intuición escribiendo código pequeño pero completo.
* Mantener trazabilidad de mi progreso sin llenar GitHub de repos aislados.

## Estructura

Cada mini-proyecto vive en su propia carpeta:

```
mini_proyectos/
    gestion_biblioteca_cli/
    gestor_notas_ui/
```

Cada carpeta contiene su propio código, datos y README si lo necesita.

## Proyectos incluidos

* **Gestor de Biblioteca CLI**: Gestión de libros y usuarios desde consola, con persistencia en JSON y lógica modular.
* **Gestor de Notas (Tkinter)**: Aplicación gráfica simple para crear, editar y guardar notas, con temas visuales y persistencia en archivo.

## Cómo usar este repositorio

* Entra en cualquier carpeta para ver el código y su README específico.
* Ejecuta cada proyecto de forma independiente según sus instrucciones.

## Notas

* Todos los proyectos intentan mantenerse autónomos entre sí.
* No se incluyen dependencias externas salvo que el proyecto lo requiera claramente.
* Cada proyecto documenta lo mínimo necesario para poder ejecutarlo o entenderlo.
